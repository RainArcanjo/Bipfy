# Bipfy

Sistema web para otimização de picking e packing de pedidos Mercado Livre Full.

## Por que criei isso?

Trabalhando com logística do Mercado Livre Full, percebi que faltava uma ferramenta simples para conferir produtos via código de barras. A solução oficial era complicada demais para o dia a dia do depósito, então resolvi criar algo prático que funcionasse direto no navegador.

## O que faz

- Importa a listagem completa de envios FULL do Mercado Livre
- Permite conferência por código de barras
- Exporta relatórios em CSV
- Funciona no celular e computador (só precisa de um navegador)

## Como usar

Você tem duas opções:

1. **Demo Online**: Acesse [bipfy.vercel.app](https://bipfy.vercel.app)
2. **Local**: Baixe o projeto e abra o `index.html` no navegador

- Para importar os produtos do mercadolivre, é necessário salvar a listagem do seu envio full como HTML e importar ao Bipfy.

## Tecnologias

Mantive simples de propósito:
- HTML5
- CSS
- JavaScript vanilla

Nada de frameworks pesados - a ideia era algo leve que rodasse em qualquer dispositivo do depósito.
